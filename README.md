## <a name="quick-start">ğŸ¤¸ Quick Start</a>

Follow these steps to set up the project locally on your machine.

**Prerequisites ğŸ“‹**

You'll need [Git](https://git-scm.com), [VSCode](https://code.visualstudio.com/) installed on your computer.

```
git@latest
code@latest
```

---

**Cloning the Repository**

From your command line, first clone this repo:

```bash
$ git clone https://github.com/ShahJabir/ferrari-clone
```

```bash
$ cd ferrari-clone
```

**Installation and Running the Project**

```bash
$ git remote remove origin
```

After that, you can use it in vscode.

You can set a server, go to that url to see the projects locally.
